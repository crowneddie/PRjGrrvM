# 项目简介

## 核心功能
- 基于 YOLOv8 的土壤微生物检测系统，支持识别 6 类微生物：Ciliates、Fungai、Microalgae、Nematode、Rotifer、flagella
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
- 提供实时检测功能，显示目标位置、目标总数和置信度信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 支持选择显示标签和原图
- 支持批量检测功能，可查看每一张检测结果
- 提供检测结果保存功能

## 适用场景
- 土壤微生物学研究与检测
- 农业生态环境监测
- 微生物多样性分析
- 实验室微生物快速筛查
- 生态学研究中的微生物计数与分类

## 技术特点
- 使用 Python 3.9 开发，依赖 OpenCV、PyQt5、Ultralytics、PyTorch 1.9 等库
- 提供 PyCharm+Anaconda 或 VSCode+Anaconda 两种开发环境选择
- 包含完整数据集（2200 张标注图片）、训练好的模型及训练结果
- 提供完整的 UI 界面，支持交互式操作
- 包含项目运行教程，便于快速部署使用
